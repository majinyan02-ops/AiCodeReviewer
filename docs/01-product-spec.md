# AI Code Reviewer（AI代码审查与自动修复平台）

## 1. 项目背景

在企业开发过程中，代码Review依赖高级工程师人工检查。

人工Review存在以下问题：

- Review效率低
- Review标准不统一
- 重复问题频繁出现
- 新人代码质量难以保证

本项目通过规则引擎 + Spring AI + 静态代码分析技术，实现自动代码审查与修复建议生成，提高代码质量和开发效率。

------

## 2. 项目目标

构建一个企业级AI代码审查平台。

支持：

- Git项目上传
- Java代码扫描
- 静态规则检测
- AI问题解释
- AI修复建议
- 审查报告生成

------

## 4. MVP功能范围

### 用户模块

- 登录
- 注册
- JWT认证

------

### 项目管理模块

创建项目

字段：

- 项目名称
- Git仓库地址
- 项目描述

支持：

- Git仓库同步
- 项目列表查询

------

### 代码扫描模块

支持扫描：

- Java项目
- SpringBoot项目

支持解析：

- Controller
- Service
- Repository
- Entity

------

### 规则检测模块

V1内置规则：

RULE-001

缺少@Transactional

RULE-002

缺少日志记录

RULE-003

捕获Exception后未处理

RULE-004

存在System.out.println

RULE-005

方法长度超过100行

RULE-006

Controller直接操作数据库

------

### AI分析模块

功能：

根据规则检测结果生成：

- 问题描述
- 风险分析
- 修复建议

输出格式：

Markdown

------

### 审查报告模块

生成：

- 项目名称
- 审查时间
- 问题列表
- 风险等级
- AI建议

支持导出PDF
